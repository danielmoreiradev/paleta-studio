# 🎨 Paleta Studio

Uma ferramenta web simples e prática para criar, gerenciar e exportar cores, gradientes e paletas de cores prontas.

## 📌 O que é Paleta Studio?

Paleta Studio é um gerador de cores totalmente online que funciona no navegador. Você:
- ✅ Escolhe cores e ajusta a transparência
- ✅ Cria gradientes personalizados (linear ou radial)
- ✅ Extrai cores de imagens automaticamente
- ✅ Organiza paletas por projeto ou cliente
- ✅ Copia o código pronto para colar no CSS do Elementor

Tudo sem instalar nada. Só abre no navegador e já funciona!

- Acesse na integra: https://danielmoreiradev.github.io/paleta-studio/paleta-studio.html
---

## 🚀 Como usar

### 1️⃣ **Abrir a ferramenta**
Acesse o arquivo `paleta-studio.html` no navegador ou acesse a versão online.

### 2️⃣ **Escolher a cor**
Na aba **"Cor & Opacidade"**:
- Clique no quadrado colorido para escolher com o seletor
- Ou digite o código HEX da cor (ex: `#38BDF8`)
- Ajuste os sliders para RGB, HSL ou opacidade
- Veja a cor em tempo real no painel à direita

### 3️⃣ **Criar um gradiente**
Na aba **"Gradiente"**:
- Escolha o tipo: **Linear** ou **Radial**
- Ajuste o ângulo (para linear) ou a posição (para radial)
- Adicione cores nas "paradas" do gradiente
- Digite o nome da classe CSS
- Copie o código CSS gerado

### 4️⃣ **Extrair cores de uma imagem**
Na aba **"Extrair de Imagem"**:
- Faça upload de uma foto, print ou imagem de referência
- A ferramenta detecta automaticamente as cores principais
- Escolha qual cor adicionar à sua paleta
- Ou crie uma paleta inteira com as cores encontradas

### 5️⃣ **Organizar suas paletas**
Na aba **"Paletas"**:
- Crie uma paleta com o nome do seu projeto ou cliente
- Adicione cores com nomes fáceis de lembrar (ex: "Verde Principal", "Cinza Claro")
- Ajuste a opacidade de cada cor
- Exporte como CSS, SCSS, JSON, TXT, HTML ou imagem PNG/JPG

---

## 💾 Salvamento automático

Todas as suas paletas são **salvas automaticamente** no navegador. Feche e abra de novo que tudo continua lá!

---

## 📥 Como exportar suas cores

Clique em **"Exportar ▾"** em qualquer paleta e escolha:

| Formato | Uso |
|---------|-----|
| **CSS** | Copiar direto para o CSS personalizado do Elementor |
| **SCSS** | Para projetos que usam SCSS |
| **JSON** | Para compartilhar ou usar em outras ferramentas |
| **TXT** | Texto simples com as cores listadas |
| **HTML** | Página com a paleta para visualizar ou imprimir |
| **PNG/JPG** | Imagem para compartilhar com o cliente |

## 🎯 Casos de uso

✨ **Designer criando paletas para cliente**
- Monte a paleta no Paleta Studio
- Exporte como HTML para mostrar o cliente
- Exporte como CSS para colar direto no Elementor

✨ **Agência com múltiplos projetos**
- Crie uma paleta para cada cliente
- Salve tudo no navegador
- Sempre disponível para reutilizar

✨ **Extrair cores de marca visual**
- Envie a logo ou imagem da marca
- Extraia as cores principais automaticamente
- Organize em uma paleta pronta

## 🛠 Recursos técnicos

- **Linguagem:** HTML5 + CSS3 + JavaScript puro (vanilla)
- **Sem dependências:** Funciona offline, sem bibliotecas externas
- **Responsivo:** Funciona em desktop, tablet e mobile
- **Armazenamento:** Usa `localStorage` do navegador para salvar paletas
